\subsection{IUE02 Saludo de personal de seguridad}

\subsubsection{Objetivo}
Mostrar una pantalla de inicio después de iniciar sesión y enumerar las acciones disponibles para el personal de seguridad.

\subsubsection{Diseño}
Esta pantalla \IUref{IUE02}{Pantalla de saludo del personal de seguridad} (ver figura~\ref{IUE02}) aparece tras un inicio de sesión exitoso y muestra las acciones específicas del personal de seguridad, además de las opciones generales del usuario (consultar calendario escolar y notificaciones).

\IUfig[.35]{IUE02}{IUE02}{Pantalla de Menú del personal de seguridad.}

\subsubsection{Salidas}
Ninguna.

\subsubsection{Entradas}
Ninguna.

\subsubsection{Comandos}
\begin{itemize}
	\item \IUbutton{Consultar alumno}: Redirige al personal de seguridad a la pantalla \IUref{IU12}{Buscar alumno}.
	\item \IUbutton{Escanear credencial}: Redirige al personal de seguridad a la pantalla \IUref{IU10}{Pantalla de código QR}.
	\item \IUbutton{Notificaciones}: Redirige al personal de seguridad a la pantalla \IUref{IU03}{Consultar notificaciones}.
	\item \IUbutton{Calendario}: Redirige al personal de seguridad a la pantalla \IUref{IU02}{Consultar calendario escolar}.
\end{itemize}

\subsubsection{Mensajes}
\begin{itemize}
	\item No aplica para esta pantalla.
\end{itemize}
